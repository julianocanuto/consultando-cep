
# [Mão na massa] - Consumindo API de CEP (ViaCep) usando OpenFeign

O objetivo foi criar uma rota que recebesse um CEP e retornasse os dados da consulta à API do ViaCep.

## Como rodar diretamente no terminal

```
mvn spring-boot:run
```
